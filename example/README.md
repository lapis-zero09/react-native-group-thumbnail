# react-native-group-thumbnail example app

## Install

### Install node modules

```
yarn
```

or

```
npm install
```

### Install react-native-group-thumbnail

```
yarn build
```

or

```
npm run build
```

### Run app

```
yarn ios
```

or

```
npm run ios
```
